Curriculum

Complete Course Curriculum 📚

4 progressive modules taking your child from complete beginner to confident coder.

1

🎨 Scratch & Block Coding

Perfect for beginners aged 6–10 — visual drag-and-drop programming

Weeks 1–4
▼
Introduction to Scratch interface
Sprites, stages and backdrops
Motion and look blocks
Events and control blocks
Loops and conditionals
Sound and music blocks
Creating animations
Simple interactive stories
🛠️ Module Project

"Space Adventure Game" — A fully playable 2-level Scratch game with score, lives, and sound effects

2

🐍 Python Fundamentals

Real text-based programming for ages 10+ — the world's favourite language

Weeks 5–10
▼
Python syntax and setup
Variables and data types
Input/output operations
Operators and expressions
Conditional statements (if/else)
Loops (for, while)
Functions and scope
Lists, dictionaries, tuples
Basic file operations
Error handling basics
🛠️ Module Project

"Snake Game in Python" — A fully functional console Snake game using core Python concepts

3

🌐 Web Development

Design and build real websites — for ages 12+ — HTML, CSS & JavaScript

Weeks 11–16
▼
HTML structure and tags
CSS styling and selectors
Flexbox and Grid layouts
Responsive design basics
JavaScript fundamentals
DOM manipulation
Events and interactivity
Forms and validation
Introduction to APIs
Deploying websites online
🛠️ Module Project

"Personal Portfolio Website" — A fully responsive, live website with projects page, contact form and animations

4

📱 App Development

Build mobile-friendly apps — for ages 14+ — React basics & UI design

Weeks 17–22
▼
Introduction to React
Components and props
State management basics
UI/UX design principles
Building interactive UIs
API integration basics
Local storage and data
App testing basics
Publishing your app
🛠️ Module Project

"Quiz & Flashcard App" — A full React app with categories, score tracking, and shareable results link
